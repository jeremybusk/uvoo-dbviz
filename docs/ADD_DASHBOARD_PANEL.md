# Dashboard Panel

##  How to add a dashboard and panel:

1. Open the **Dashboards** section in the left sidebar.
2. Click **New**.
3. Enter a dashboard name.
4. Configure the query in the **Query** section: dataset, group, filters, time range, visualization, etc.
5. Go back to **Dashboards**.
6. Set **Panel title** and **Visualization**.
7. Click **Add**. This adds the current query as a panel.
8. Repeat query changes + **Add** for more panels.
9. Click **Save** to persist the dashboard.

To edit an existing panel:

1. Open the dashboard.
2. Click the panel in the staged panel list or click **Edit** on the panel card.
3. Change query/title/visualization.
4. Click **Update**.
5. Click **Save** dashboard.
